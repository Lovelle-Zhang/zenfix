# Zenfix 签文库重构计划

## 目标
每个人格20条签文，覆盖6个类别：事业、感情、健康、财运、人际、成长

## 类别分布（每个人格）
- 事业：4条
- 感情：3条  
- 健康：3条
- 财运：3条
- 人际：3条
- 成长：4条

总计：20条

## 实施策略
由于当前每个人格已有20条签文（只覆盖3个类别），需要：
1. 保留每个类别的精华签文
2. 删除部分重复的签文
3. 添加缺失类别的签文

## 具体调整

### ZenClaw
**保留**：
- 事业：0x01, 0x02, 0x07, 0x0A（4条）
- 感情：0x03, 0x04, 0x08（3条）
- 成长：0x05, 0x06, 0x09（3条）

**新增**：
- 健康：0x15, 0x16, 0x17（3条）
- 财运：0x18, 0x19, 0x1A（3条）
- 人际：0x1B, 0x1C, 0x1D（3条）

**删除**：0x0B-0x14（10条，重复类别）

### 其他人格
采用相同逻辑调整

## AI解签示例覆盖
为每个类别创建至少1个AI解签示例：
- 4个人格 × 6个类别 = 24个AI解签示例
- 覆盖率：24/80 = 30%
