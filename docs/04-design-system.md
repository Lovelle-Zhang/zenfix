# Zenfix UI/UX 设计系统

## 一、设计哲学：Biophilic Future (亲生物未来)

**核心理念**：将数字体验与自然元素深度融合，创造一种"置身森林深处"的静谧感。

### 1.1 视觉基因 (Visual DNA)

从森林中提取的四大视觉元素：

1. **光斑 (Bokeh)**
   - 柔和的失焦光点
   - 营造深度和层次感
   - 动态呼吸效果

2. **苔藓绿 (Moss Green)**
   - 主色调：#4A6741, #5F7C57, #7A9971
   - 辅助色：#3D5A3A, #8FAF88
   - 传递生命力和平静感

3. **有机呼吸感 (Organic Pulse)**
   - 所有动画都模拟自然节奏
   - 避免机械的线性运动
   - 使用缓动函数：ease-in-out-sine

4. **晨雾 (Mist)**
   - 半透明的渐变层
   - 柔化边界
   - 营造神秘感

### 1.2 反设计原则

**避免的风格**：
- ❌ 赛博朋克（过于冷硬）
- ❌ 极简主义（过于冷淡）
- ❌ 新拟物化（过于做作）
- ❌ 传统中国风（过于陈旧）

**追求的感觉**：
- ✅ 温暖但不甜腻
- ✅ 神秘但不诡异
- ✅ 科技但不冰冷
- ✅ 自然但不原始

## 二、色彩系统

### 2.1 主色板

```
主色（Primary）：
- Moss Green #5F7C57
- Deep Forest #3D5A3A
- Light Moss #8FAF88

辅助色（Secondary）：
- Twilight Blue #4A5F7C (夜幕蓝)
- Amber Glow #D4A574 (琥珀光)
- Mist White #E8EDE8 (雾白)

功能色（Functional）：
- Success: #7A9971
- Warning: #D4A574
- Error: #A67C6D (柔和的红褐)
- Info: #4A5F7C
```

### 2.2 渐变系统

**主渐变（用于背景）**：
```css
background: linear-gradient(
  135deg,
  #3D5A3A 0%,
  #4A6741 50%,
  #5F7C57 100%
);
```

**光斑渐变（用于装饰）**：
```css
background: radial-gradient(
  circle at 30% 40%,
  rgba(143, 175, 136, 0.3) 0%,
  transparent 70%
);
```

## 三、字体系统

### 3.1 中文字体

**标题**：思源宋体 / Source Han Serif
- 优雅、有文化感
- 适合哲学性内容

**正文**：苹方 / PingFang SC
- 清晰易读
- 现代感

**强调**：霞鹜文楷 / LXGW WenKai
- 手写感
- 温暖、人性化

### 3.2 英文字体

**标题**：Playfair Display
- 优雅的衬线体
- 适合标题和引用

**正文**：Inter
- 现代无衬线
- 极佳的可读性

**等宽**：JetBrains Mono
- 用于技术信息
- 清晰的字符区分

### 3.3 字号系统

```
H1: 32px / 2rem (页面标题)
H2: 24px / 1.5rem (章节标题)
H3: 20px / 1.25rem (小节标题)
Body: 16px / 1rem (正文)
Small: 14px / 0.875rem (辅助信息)
Tiny: 12px / 0.75rem (标签、时间戳)
```

## 四、交互设计

### 4.1 核心交互流程

```
启动App
  ↓
[15秒降噪模式]
- 粒子流动画
- 引导深呼吸
- 背景音：432Hz
  ↓
[生物反馈采集]
- "请将手指轻放在屏幕上"
- 长按屏幕（3-5秒）
- 实时显示"量子纠缠"可视化
  ↓
[签文生成]
- 粒子汇聚动画
- "正在观测您的量子态..."
- 签文卡片翻转出现
  ↓
[AI解签]
- 选择人格（默认：赛博禅宗）
- AI生成解签内容
- 显示行动建议
  ↓
[艺术生成]
- "正在为您绘制数字护身符..."
- 生成式艺术图渐显
- 可保存、分享、定制
  ↓
[互助系统]（可选）
- 显示全球共振数据
- 可留下光点
- 可接收他人的光点
```

### 4.2 手势交互

**长按** - 生物反馈采集  
**上滑** - 查看历史记录  
**下滑** - 刷新/重新抽签  
**双指捏合** - 放大艺术图  
**左右滑动** - 切换AI人格

### 4.3 动画原则

**自然节奏**：
- 所有动画时长：300-800ms
- 缓动函数：ease-in-out-sine（模拟自然运动）
- 避免突兀的开始和结束

**粒子系统**：
- 使用Canvas或WebGL
- 粒子数量：500-1000（根据设备性能调整）
- 运动模式：布朗运动 + 引力场

**过渡效果**：
- 淡入淡出（Fade）
- 模糊过渡（Blur）
- 避免硬切（Hard Cut）

## 五、组件设计

### 5.1 签文卡片 (Oracle Card)

**尺寸**：屏幕宽度的85%，高度自适应

**结构**：
```
┌─────────────────────────────┐
│   [签文类别]  [签文等级]     │
│                              │
│      签文内容（大字）         │
│                              │
│   关键词：#焦虑 #迷茫 #希望  │
│                              │
│   [查看解签] [重新抽签]      │
└─────────────────────────────┘
```

**视觉效果**：
- 毛玻璃背景（backdrop-filter: blur(20px)）
- 柔和阴影
- 边缘发光（box-shadow: 0 0 20px rgba(...)）

### 5.2 AI解签面板 (Interpretation Panel)

**布局**：全屏模态

**内容区块**：
1. **人格标识**（顶部）
   - 人格名称 + 图标
   - 切换按钮

2. **解签内容**（中部）
   - 哲学慰藉（主要内容）
   - 情感分析
   - 认知重构

3. **行动建议**（底部）
   - 具体可执行的微小行动
   - 预计时长
   - 完成按钮

### 5.3 数字护身符 (Digital Talisman)

**展示方式**：
- 全屏沉浸式
- 可左右滑动查看细节
- 长按保存到相册

**元数据显示**：
- 生成时间
- 量子种子（伪随机数种子）
- 情绪标签

## 六、音效设计

### 6.1 背景音乐

**疗愈频率**：
- 432Hz（自然和谐频率）
- 528Hz（DNA修复频率）

**音色选择**：
- 合成器Pad（持续音）
- 环境音（森林、海浪、风声）
- 避免旋律性过强

### 6.2 交互音效

**触摸反馈**：
- 轻柔的"叮"声（水滴音）
- 音量：30-40%

**粒子流动**：
- 白噪音变体
- 营造"量子场"的感觉

**签文出现**：
- 钟声 / 磬声
- 庄重但不沉重

## 七、响应式设计

### 7.1 屏幕适配

**小屏（< 375px）**：
- 减少粒子数量
- 简化动画
- 优先显示核心内容

**中屏（375-414px）**：
- 标准体验
- 完整动画效果

**大屏（> 414px / 平板）**：
- 增加视觉细节
- 更丰富的粒子效果
- 可显示更多上下文信息

### 7.2 性能优化

**低端设备**：
- 自动降低粒子数量
- 使用CSS动画替代Canvas
- 预加载关键资源

**高端设备**：
- 启用高级视觉效果
- 60fps流畅动画
- 实时艺术生成

## 八、无障碍设计 (Accessibility)

### 8.1 视觉无障碍

- 色彩对比度 ≥ 4.5:1（WCAG AA标准）
- 支持系统字体大小调整
- 提供高对比度模式

### 8.2 操作无障碍

- 所有交互都有语音提示（可选）
- 支持VoiceOver / TalkBack
- 触摸目标 ≥ 44x44px

### 8.3 认知无障碍

- 清晰的信息层级
- 避免过度复杂的交互
- 提供"新手引导"模式
