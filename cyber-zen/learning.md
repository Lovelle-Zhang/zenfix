## 语言风格学习

### 禅宗公案的代码化

**原公案**: 赵州问："如何是祖师西来意？" 南泉答："庭前柏树子。"
**赛博化**: 
```
> QUERY: 系统的初始化意图是什么？
> RESPONSE: /dev/random
```

**原公案**: "不思善，不思恶，正与么时，哪个是明上座本来面目？"
**赛博化**:
```
> EXEC: unset $good && unset $evil
> QUERY: echo $self
> RESPONSE: null
```

## 签文创作模板

### 结构
1. 四字或短句（意象）
2. 一句话本质（否定式）
3. 技术隐喻展开
4. 行为指令（极简）
5. 代码式执行

### 创作原则
- 不安慰，只揭示
- 不指导，只断链
- 不解释，只点破
- 不温暖，只清醒
